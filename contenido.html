Operaciones Basicas

Suma: La suma es una operación matemática básica que combina dos o más números para obtener un total o una cantidad. Se representa con el símbolo "+" y se lee como "más". Por ejemplo, en la suma 2 + 3 = 5, se están combinando los números 2 y 3 para obtener 5 como resultado.

Resta: La resta es otra operación básica que consiste en encontrar la diferencia entre dos números. Se representa con el símbolo "-" y se lee como "menos". Por ejemplo, en la resta 5 - 3 = 2, se está restando 3 de 5 para obtener 2 como resultado.

Multiplicación: La multiplicación es una operación que combina grupos iguales de números para encontrar el total. Se representa con el símbolo "x" o con un punto "." y se lee como "por" o "multiplicado por". Por ejemplo, en la multiplicación 2 x 3 = 6, se están multiplicando 2 grupos de 3 para obtener 6 como resultado.

División: La división es la operación inversa de la multiplicación y se utiliza para repartir una cantidad en grupos iguales o para encontrar cuántas veces un número cabe en otro. Se representa con el símbolo "÷" o con una línea horizontal y se lee como "dividido por". Por ejemplo, en la división 6 ÷ 3 = 2, se está dividiendo 6 en 3 grupos iguales para obtener 2 como resultado.
